# 场景独立记录：危重症 - 严重口渴（非慢病）

## 场景标识

- **大类**：危重症等紧急情况
- **子类**：非慢病的危急症状
- **症状**：严重口渴

## 应用规范摘要

符合急危重症时：不进一步追问，根据已有描述选用对应话术，将患者不适带入话术应答。

## 匹配条件要点

- **伴随/关联高危**：意识异常、晕倒、皮肤湿冷苍白或发花、面色苍白发灰发紫、脉搏细速微弱或难以摸到、呼吸频率异常、低血压或SBP下降＞40、四肢冰凉、大汗、呼气烂苹果味、进行性加重乏力、严重呕吐腹泻、剧烈腹痛、无尿、皮肤弹性极差、抽搐、严重头痛、呼吸困难、心率快或乱跳、胸痛、高热、突发难以解释的严重口渴等。
- **无高危**：服用锂盐/抗胆碱能药后出现、慢性肾病/心衰患者伴随严重口渴且病情未加重、口干+眼干+肌痛、高血压+肌无力肢端麻木手足抽搐多尿等。

## 回复话术

看到你出现了严重口渴的情况，需要引起重视。严重口渴的原因多种多样，可能涉及内分泌疾病、风湿免疫或其他疾病，因此建议你在家人陪同下立即前往医院就诊，以明确原因。另外，因线上沟通存在局限性，以上建议仅为初步指导，就医后，医生会根据你的具体情况给予针对性的建议。
