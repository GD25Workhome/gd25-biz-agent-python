# 场景独立记录：危重症 - 粉红色泡沫痰

## 场景标识

- **大类**：危重症等紧急情况
- **子类**：慢病及其并发症相关的危急症状
- **症状**：粉红色泡沫痰（肺泡内液体渗出与空气混合成泡沫，混有红细胞呈粉红色）

## 应用规范摘要

符合急危重症时：不进一步追问，根据已有描述选用对应话术，将患者不适带入话术应答。本条分两种话术分支。

## 匹配与话术分支

### 分支一：首次出现或大量粉红色泡沫痰

**匹配**：首次出现粉红色泡沫样痰、或大量粉红色泡沫痰，无论是否伴随其他症状。

**回复话术**：你好，注意到你有粉红色泡沫痰的情况需要特别关注，不排除存在急危重症的可能，需要高度警惕，强烈建议你立即由专业医护人员评估。请你拨打120，保持舒服的体位，如坐位或半坐位，双腿下垂。同时，解开领口、腰带等，保持呼吸通畅。告知120的工作人员你目前的不适症状。另外，因线上沟通有局限性，以上建议仅为初步指导，一切以医务人员现场判断为准。后续请按医生指导，采取下一步措施。

### 分支二：非首次、量极少且已就诊

**匹配**：非首次出现的粉红色泡沫痰，量极少，且已就诊过。

**回复话术**：看到你再次出现粉红色泡沫痰，且虽已就诊，但症状仍有反复，需要引起高度重视。因此，建议你在家人陪同下立即前往医院就诊。另外，由于线上沟通存在局限性，以上仅为初步建议，就医后医生会根据你的具体情况给予针对性的指导。
