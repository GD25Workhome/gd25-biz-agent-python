# JSON 编辑器方案对比

**文档编号**：020501  
**创建日期**：2025-02-05  
**用途**：数据清洗界面中 JSON 字符串友好编辑组件的选型参考  

---

## 1. 方案概览

在 `frontend/data-cleaning.html` 的「JSON编辑器方案对比」页面中，提供了 4 种 JSON 编辑实现，可通过左侧菜单进入，切换 Tab 查看效果。

| 方案 | 组件/库 | 特点 | 体积 | 适用场景 |
|------|---------|------|------|----------|
| **01-Textarea** | Element Plus textarea | 零依赖，轻量，格式化/校验按钮 | 0 | 简单配置编辑 |
| **02-JSONEditor** | jsoneditor (josdejong) | 树形/代码/文本模式，搜索、格式化 | ~1MB | 结构化编辑、导入配置 |
| **03-Ace** | Ace Editor | 代码高亮、行号、主题 | ~300KB | 代码风格编辑 |
| **04-Monaco** | Monaco Editor (VS Code) | 智能提示、折叠、多光标 | ~2MB+ | 复杂 JSON、需 IntelliSense |

---

## 2. 方案说明

### 2.1 JSON编辑器-01-Textarea

- **实现**：`el-input type="textarea"` + 格式化、校验按钮
- **依赖**：无额外依赖
- **优点**：零依赖、实现简单、与现有 Element Plus 风格一致
- **缺点**：无语法高亮、无行号、无树形视图
- **推荐**：`import_config` 等简单 JSON 字段的快速编辑

### 2.2 JSON编辑器-02-JSONEditor

- **实现**：`jsoneditor` (josdejong)，CDN: `cdn.jsdelivr.net/npm/jsoneditor@10.4.2`
- **依赖**：jsoneditor.min.js + jsoneditor.min.css
- **优点**：树形/代码/文本模式切换，搜索、格式化、校验，功能丰富
- **缺点**：体积较大（~1MB），树形模式对深层嵌套支持一般
- **推荐**：导入配置等结构化 JSON 的友好编辑，可替代当前 textarea

### 2.3 JSON编辑器-03-Ace

- **实现**：Ace Editor，CDN: `cdn.jsdelivr.net/npm/ace-builds`
- **依赖**：ace.js
- **优点**：代码高亮、行号、主题，轻量
- **缺点**：纯代码视图，无树形
- **推荐**：偏好代码风格、需要行号时使用

### 2.4 JSON编辑器-04-Monaco

- **实现**：Monaco Editor，CDN: `cdn.jsdelivr.net/npm/monaco-editor`
- **依赖**：loader.js + AMD 加载
- **优点**：VS Code 内核，智能提示、折叠、多光标
- **缺点**：体积大、加载较慢、需 AMD 环境
- **推荐**：复杂 JSON、需要 IntelliSense 或 Schema 校验时

---

## 3. 接入建议

针对 `import_config` 等 JSON 字段的编辑场景：

1. **快速落地**：沿用 01-Textarea，增加格式化/校验按钮（已具备）
2. **体验优先**：选用 02-JSONEditor，树形模式便于非技术用户编辑（推荐）
3. **轻量优先**：选用 03-Ace

---

## 4. 接入情况

已使用 **02-JSONEditor** 方案升级以下 JSON 字段的编辑组件：

| 组件 | 字段 |
|------|------|
| Step00 导入管理 | import_config |
| Step01 原始数据管理（数据集） | input_schema、output_schema、metadata |
| 数据项管理 | input、output、metadata |

可复用组件：`frontend/js/pipeline_json_editor.js`（PipelineJsonEditor）

---

## 5. 访问方式

1. 启动后端服务
2. 打开 `http://localhost:8000/static/data-cleaning.html`
3. 点击左侧菜单「Step00导入管理」「Step01原始数据管理」或「数据项」进行新增/编辑，JSON 字段已使用树形编辑器
4. 「JSON编辑器方案对比」可切换 Tab 查看其他方案效果
