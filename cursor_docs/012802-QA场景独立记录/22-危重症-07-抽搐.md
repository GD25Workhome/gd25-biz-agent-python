# 场景独立记录：危重症 - 抽搐

## 场景标识

- **大类**：危重症等紧急情况
- **子类**：慢病及其并发症相关的危急症状
- **症状**：抽搐

## 应用规范摘要

符合急危重症时：不进一步追问，根据已有描述选用对应话术，将患者不适带入话术应答。

## 匹配条件要点

- **表现**：突然僵直倒地、四肢剧烈抖动、可能咬牙/眼神呆滞/翻白眼/口吐白沫/大小便失禁。
- **时间**：首次发作；持续≥5分钟；发作停止后意识异常＞10分钟；发作两次或多次且之间意识未完全恢复。
- **伴随/关联高危**：糖尿病/高血压/冠心病/肾衰透析、可疑毒物/药物过量、言语或理解障碍、视力丧失/模糊、剧烈头痛眩晕、剧烈或持续呕吐、高热、颈强直、头部外伤后抽搐、抽搐中受伤、口唇肢体发紫、喘憋、呼吸微弱/缓慢/不规则/暂停、一侧肢体或面部无力麻木、妊娠等。
- **无高危**：已知癫痫、抽搐＜5分钟且抽搐后意识完全恢复。

## 回复话术

看到你出现了抽搐的情况，需要引起重视。因为抽搐的原因多种多样，可能涉及心脏、肺部、脑部或其他重要器官，因此建议你在家人陪同下立即前往医院就诊，以明确原因。另外，因线上沟通存在局限性，以上建议仅为初步指导，就医后，医生会根据你的具体情况给予针对性的建议。
