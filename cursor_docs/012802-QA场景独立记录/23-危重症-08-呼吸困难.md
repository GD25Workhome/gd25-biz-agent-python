# 场景独立记录：危重症 - 呼吸困难

## 场景标识

- **大类**：危重症等紧急情况
- **子类**：慢病及其并发症相关的危急症状
- **症状**：呼吸困难

## 应用规范摘要

符合急危重症时：不进一步追问，根据已有描述选用对应话术，将患者不适带入话术应答。

## 匹配条件要点

- **特点/诱因**：进行性加重（数分钟~数小时内）；静坐休息时难以呼吸；突然“快憋死了”；端坐呼吸、夜间憋醒；外伤/手术/长途旅行/久卧久坐/溺水/异物吸入/过敏原接触后；特殊呼吸模式（两短吸一长呼、深大呼吸伴叹息）等。
- **伴随/关联高危**：呼吸频率异常、血压异常、鼻孔张大、大汗、口唇肢端发紫、全身皮疹荨麻疹、恶心呕吐腹痛、肢冷、胸痛胸闷、含服硝酸甘油无缓解、胸部不对称、粉红色泡沫痰、心率异常、头面舌喉肿胀、三凹征、单侧肢体肿胀、喘鸣、意识异常、极度虚弱、言语障碍、视力障碍、无尿、抽搐、肢体无力麻木、头痛头晕、晕倒、妊娠初次呼吸困难、情绪强烈恐惧、哮喘急救药无效、短期体重骤增等。
- **无高危**：轻度活动气短休息可缓解；日常活动时新发呼吸困难。

## 回复话术

看到你出现了呼吸困难的情况，需要引起重视。呼吸困难的原因多种多样，可能涉及气道、肺部、心脏、神经肌肉病变等，因此建议你在家人陪同下立即前往医院就诊，以明确原因。另外，因线上沟通存在局限性，以上建议仅为初步指导，就医后，医生会根据你的具体情况给予针对性的建议。
