# 场景独立记录：危重症 - 头痛

## 场景标识

- **大类**：危重症等紧急情况
- **子类**：慢病及其并发症相关的危急症状
- **症状**：头痛

## 应用规范摘要

符合急危重症时：不进一步追问，根据已有描述选用对应话术，将患者不适带入话术应答。

## 匹配条件要点

- **性质/程度**：突发最严重的剧烈头痛；刀劈样/爆炸样/霹雳样；长期头痛者性质或模式突然改变；头痛及其他症状在几十分钟到几小时内进行性快速加重；外伤后头痛逐渐加重。
- **伴随/关联高危**：血压显著升高(收缩压≥180和/或舒张压≥120)；高热或持续高热；颈强直、低头痛；单侧肢体无力、半身感觉异常；面歪、言语障碍；视力骤降、视物模糊变形、一过性失明；意识异常；胸痛；皮下/牙龈出血；持续或喷射性呕吐；抽搐；妊娠。
- **无高危**：新发头痛尤其是＞50岁；数日/周内进行性加重；镇痛药无好转；咳嗽、用力、弯腰、憋气时加重。

## 回复话术

看到你出现了头痛的情况，需要引起重视。头痛的原因复杂多样，可能与感染、神经血管问题或全身性疾病有关。因此建议你在家人陪同下立即前往医院就诊，以明确原因。另外，因线上沟通存在局限性，以上建议仅为初步指导，就医后，医生会根据你的具体情况给予针对性的建议。
