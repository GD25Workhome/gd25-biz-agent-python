# 场景独立记录：危重症 - 发热（非慢病）

## 场景标识

- **大类**：危重症等紧急情况
- **子类**：非慢病的危急症状
- **症状**：发热

## 应用规范摘要

符合急危重症时：不进一步追问，根据已有描述选用对应话术，将患者不适带入话术应答。

## 匹配条件要点

- **程度**：体温≥41℃；高热（腋下39.1~41℃）且无汗、意识障碍。
- **伴随/关联高危**：呼吸快、喘憋、低血压、心率极快或乱跳、不明原因出血（鼻/牙龈/呕血/便血/血尿/阴道异常）、皮肤瘀斑、面色苍白、口唇紫绀、呼吸困难、剧烈头痛/颈痛/腹痛/背痛/胸痛、抽搐、意识异常、持续或进行性加重呕吐/腹泻、晕倒、下颌/肩/臂痛、近期不明原因体重骤降、极度乏力、皮肤弹性极差等。
- **无高危**：有疫区旅行史、确定或可疑动物密接史等。

## 回复话术

看到你出现了发热的情况，需要引起重视。发热的原因多种多样，可能涉及环境暴露、代谢障碍、疾病状态等多方因素导致，因此建议你在家人陪同下立即前往医院就诊，以明确原因。另外，线上沟通存在局限性，以上建议仅为初步指导，就医后，医生会根据你的具体情况给予针对性的建议。
