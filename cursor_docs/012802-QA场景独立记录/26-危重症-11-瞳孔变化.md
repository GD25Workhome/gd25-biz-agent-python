# 场景独立记录：危重症 - 瞳孔变化

## 场景标识

- **大类**：危重症等紧急情况
- **子类**：慢病及其并发症相关的危急症状
- **症状**：瞳孔变化

## 应用规范摘要

符合急危重症时：不进一步追问，根据已有描述选用对应话术，将患者不适带入话术应答。

## 匹配条件要点

- **定义与表现**：瞳孔不等大（单侧明显大于对侧如超2mm）；针尖样瞳孔（＜1mm）；瞳孔散大（＞5mm）。
- **程度/性质**：突发瞳孔不等大；针尖样瞳孔；瞳孔散大、固定。
- **伴随/关联高危**：外伤史、可疑药物接触/过量服药史、光反射减弱或消失、突发剧烈头痛、突发呼吸困难、意识异常、言语障碍、高热等。
- **无高危**：眼睛胀痛、视力下降、眼睑下垂、颈部疼痛、光线变化时瞳孔差异明显等。

## 回复话术

看到你出现了瞳孔变化的情况，需要引起重视。瞳孔变化的原因多种多样，可能由脑炎、脑疝、青光眼或其他疾病导致，因此建议你在家人陪同下立即前往医院就诊，以明确原因。另外，因线上沟通存在局限性，以上建议仅为初步指导，就医后，医生会根据你的具体情况给予针对性的建议。
