## 单条记录结构说明

每条 md 一般包含：

- **场景标识**：大类、子类/子场景
- **大类标准** 或 **应用规范摘要**：该场景的边界与使用规则
- **患者问题示例** 或 **匹配条件要点**：用于判定是否命中该场景
- **回复话术** 或 **回复话术要求**：固定话术或可灵活套用的要求

后续可将任意一条记录的全文作为输入，交给提示词做：场景摘要、问题优化、回复案例抽取与优化。
