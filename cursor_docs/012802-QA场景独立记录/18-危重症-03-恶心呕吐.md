# 场景独立记录：危重症 - 恶心、呕吐

## 场景标识

- **大类**：危重症等紧急情况
- **子类**：慢病及其并发症相关的危急症状
- **症状**：恶心、呕吐

## 应用规范摘要

符合急危重症时：不进一步追问，根据已有描述选用对应话术，将患者不适带入话术应答。

## 匹配条件要点

- **性质**：喷射状呕吐；呕吐物粪臭味；呕血（粉红/鲜红/咖啡/黑色）；呕出绿色胆汁样物。
- **伴随/关联高危**：剧烈头痛、颈强直、胸痛、腰背撕裂痛、视野缺损/复视、剧烈持续腹痛、黑便/柏油便、高热、头部外伤后呕吐、持续性腹胀不放屁不排便、低血压、心动过缓、面色白/紫、口唇面部水肿、意识异常、荨麻疹+全身瘙痒、喘憋、呼吸快、烂苹果味、半身感觉异常、言语障碍、重度脱水表现等。
- **无高危**：恶心呕吐＞24小时未缓解、进行性加重、无法进食水+心率＞100、恶心呕吐+头晕或眩晕。

## 回复话术

看到你出现了恶心呕吐的情况，需要引起重视。恶心呕吐的原因多种多样，可能涉及消化系统、神经系统、代谢问题、药物副作用等因素。因此建议你在家人陪同下立即前往医院就诊，以明确原因。另外，因线上沟通存在局限性，以上建议仅为初步指导，就医后，医生会根据你的具体情况给予针对性的建议。
