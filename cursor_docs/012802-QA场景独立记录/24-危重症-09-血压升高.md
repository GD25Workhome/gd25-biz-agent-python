# 场景独立记录：危重症 - 血压升高

## 场景标识

- **大类**：危重症等紧急情况
- **子类**：慢病及其并发症相关的危急症状
- **症状**：血压升高

## 应用规范摘要

符合急危重症时：不进一步追问，根据已有描述选用对应话术，将患者不适带入话术应答。血压显著升高但未达180/120若已出现伴随高危，也按本场景处理。

## 匹配条件要点

- **定义**：短期内高压≥180和/或低压≥120；或收缩压≥220和/或舒张压≥140。
- **伴随/关联高危**：突发言语障碍、面歪、一侧肢体无力、意识异常、抽搐、剧烈或进行性加重头痛、与体位无关的严重头晕、恶心呕吐、胸部撕裂/刀割/压迫样疼痛及放射痛、颈强直、呼吸异常、口唇肢端发紫、心率异常、胸闷胸痛≥10分钟、粉红色泡沫痰、咽部紧缩感、烦躁、冷汗、脉搏弱、无尿少尿、下肢水肿、晕倒、大小便失禁、瞳孔不等大、视力障碍、乏力、皮肤瘀点瘀斑、鼻血、呕血、黑便、血尿、头部外伤、妊娠、情绪强烈恐惧等。
- **无高危**：测血压≥2次达180/120且无上述不适。

## 回复话术

看到你出现了血压快速升高的情况，需要引起重视。长期血压波动性升高可能会导致心脏、脑部、肾脏等重要器官的损害，因此建议你在家人陪同下立即前往医院就诊。另外，因线上沟通存在局限性，以上建议仅为初步指导，就医后，医生会根据你的具体情况给予针对性的建议。
