# 场景独立记录：危重症 - 偏瘫

## 场景标识

- **大类**：危重症等紧急情况
- **子类**：慢病及其并发症相关的危急症状
- **症状**：偏瘫（一侧肢体无力/不能活动或一侧身体麻木）

## 应用规范摘要

符合急危重症时：不进一步追问，根据已有描述选用对应话术，将患者不适带入话术应答。

## 匹配条件要点

- **表现**：一侧肢体无力或完全不能活动（如持物掉落、行走困难、拖沓、跌倒、踩棉花感）；一侧身体麻木。
- **时间**：突发偏瘫（数秒或数分钟内达峰）。
- **伴随/关联高危**：糖尿病/高血压/冠心病/房颤等、瞳孔不等大、突发呼吸困难、突发胸痛、血压≥180/120、意识异常、剧烈头痛、言语障碍、视力障碍、面歪麻木等。
- **无高危**：一过性偏瘫、缓慢进展、确诊癫痫抽搐后肢体无力。

## 回复话术

看到你出现肢体变化的情况，需要引起重视。因为肢体变化的原因多种多样，如脑卒中、脑外伤、脑部肿瘤等，因此建议你在家人陪同下立即前往医院就诊，以明确原因。另外，因线上沟通存在局限性，以上建议仅为初步指导，就医后，医生会根据你的具体情况给予针对性的建议。
