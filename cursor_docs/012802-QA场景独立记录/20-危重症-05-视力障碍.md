# 场景独立记录：危重症 - 视力障碍

## 场景标识

- **大类**：危重症等紧急情况
- **子类**：慢病及其并发症相关的危急症状
- **症状**：视力障碍

## 应用规范摘要

符合急危重症时：不进一步追问，根据已有描述选用对应话术，将患者不适带入话术应答。

## 匹配条件要点

- **表现**：视力模糊；视野缺损；复视；视力丧失（无光感）。
- **时间/性质**：单眼或双眼突发视力急剧下降或丧失；眼前固定黑影、闪光感；短暂性视力丧失/视物模糊/视野缺损（1–15分钟）。
- **伴随/关联高危**：面歪、言语障碍、半身感觉异常、剧烈头痛头晕、恶心呕吐、胸痛胸闷、收缩压≥180和/或舒张压≥120。
- **无高危**：缓慢进展的视力下降（老花、白内障、近视加深）；短暂视物模糊（数秒~几分钟）；突发眼胀痛；眼部外伤后视力障碍；瞳孔浑浊、发热、眼红等。

## 回复话术

看到你出现了视力障碍的情况，需要引起重视。因为视力障碍的原因多种多样，可能与眼部、脑部或全身性疾病等因素有关。因此建议你在家人陪同下立即前往医院就诊，以明确原因。另外，因线上沟通存在局限性，以上建议仅为初步指导，就医后，医生会根据你的具体情况给予针对性的建议。
