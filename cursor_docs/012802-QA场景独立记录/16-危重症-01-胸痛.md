# 场景独立记录：危重症 - 胸痛

## 场景标识

- **大类**：危重症等紧急情况
- **子类**：慢病及其并发症相关的危急症状
- **症状**：胸痛

## 应用规范摘要

符合急危重症时：不进一步追问，根据已有描述选用对应话术，将患者不适带入话术应答；症状带入时括号内文案不加入。

## 匹配与话术分支

### 分支一：冠心病用户

**匹配**：胸痛 + 冠心病病史。

**回复话术**：鉴于您的病史（冠心病），请及时联系医生！！拨打24小时电话021-6666666

### 分支二：非冠心病用户 — 需120/立即就诊

**匹配**：胸痛程度/性质（突发剧烈、难以忍受；撕裂/刀割/重物压感；持续加重）；或胸痛时间（新发；≥15分钟；2–15分钟且含服硝酸甘油无效或效果不佳；发作变多变密、单次更久）；或胸痛伴随/关联高危（活动诱发或加重、休息不缓解、心跳极快或乱跳、无力、脸色白/紫、大汗、肢冷、头晕、晕倒、言语/面歪、恶心呕吐、意识异常、低血压、肢体肿胀、血压差>20、放射痛、牙痛、喘憋、胸部不对称、咯血/呕血、高热、情绪强烈恐惧等）。

**回复话术**：依据患者特征按「紧急情况话术模板」指导拨打120或立即就诊（将“胸痛”等不适带入模板中的[症状描述]）。

### 分支三：非冠心病用户 — 无高危

**匹配**：胸痛无伴随或关联上述高危；与呼吸有关；超过一周的轻度胸痛；按压胸部疼痛加重。

**回复话术**：了解到你出现了胸痛的情况，需要引起重视。因为胸痛的原因多种多样，可能涉及心脏、肺部或其他重要器官，因此建议你在家人陪同下及时前往医院就诊，以明确原因。另外，因线上沟通存在局限性，以上建议仅为初步指导，就医后，医生会根据你的具体情况给予针对性的建议。
