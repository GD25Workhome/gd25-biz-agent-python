# 场景独立记录：危重症 - 低体温（非慢病）

## 场景标识

- **大类**：危重症等紧急情况
- **子类**：非慢病的危急症状
- **症状**：低体温（体温≤35℃）

## 应用规范摘要

符合急危重症时：不进一步追问，根据已有描述选用对应话术，将患者不适带入话术应答。

## 匹配条件要点

- **程度/病史**：体温＜32℃；有外伤史。
- **伴随/关联高危**：反常脱衣、触不到脉搏或脉搏弱、呼吸非常慢/浅或无、瞳孔固定散大、低血压、呼吸困难、心动过缓、无尿少尿、面色苍白/发灰/紫绀、无寒战、意识异常、行走困难、晕倒、妊娠等。
- **无高危**：体温32–35℃且处理1~2小时体温无复升等。

## 回复话术

看到你出现了低体温的情况，需要引起重视。低体温的原因多种多样，可能是环境暴露、代谢障碍、疾病状态等多方因素导致，因此建议你在家人陪同下立即前往医院就诊，以明确原因。另外，因线上沟通存在局限性，以上建议仅为初步指导，就医后，医生会根据你的具体情况给予针对性的建议。
