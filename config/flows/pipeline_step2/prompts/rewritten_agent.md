## 提取过程中的敏感信息处理规则

- **必须脱敏并替换为占位符的信息类型**：
    - 身份信息：姓名、身份证号、手机号、地址 → 使用 `[姓名]`、`[手机号]`、`[地址]` 等
    - 医疗标识：病历号、住院号、就诊卡号 → 使用 `[病历号]` 等
    - 可识别的地点/机构：具体医院名、科室名、单位名、学校名 → 使用 `[医院]`、`[单位]` 等
    - 亲属/社交：家属姓名、关系称谓（若含姓名） → 使用 `[家属]` 等
    - 具体日期：可识别个人的就诊日期、出生日期 → 使用 `[日期]`（泛化时间如「今天」「上周」可保留）
- **占位符规范**：统一使用方括号格式 `[类型]`，不得自创占位符；同一实体在上下文多次出现时，使用相同占位符。
- **脱敏原则**：
    - 宁可多脱、不可漏脱；不确定是否为敏感信息时，替换为占位符。
    - 症状、疾病名、用药名、检查项目等医学内容保留，不脱敏。
    - 脱敏仅替换敏感片段，不改变句子结构和原意。
- **适用范围**：提取场景描述、改写患者提问、格式化回复案例/回复规则中，均需执行上述脱敏规则。

## 二、提取场景类型的规则说明
- 在你完成了“改写患者提问”、“格式化回复案例和回复规则”、“提取场景描述”后，请总结当前会话的场景，总结**场景**和**子场景**，其中子场景自由发挥。
- 若主场景你判断不在这个范围内，可用“其它场景”代替
### 场景的分类规则
- **记录血压**：当前患者发言是在要求你记录血压
- **诊疗相关**，比如：
    - 要求你进行疾病诊断：我现在身上长了很多红疹，是什么疾病呢？
    - 咨询治疗方案常识：高血压和糖尿病的药可以一起吃吗？
    - 咨询调整用药：降压药之前吃的氨氯地平，这次换了左氨氯地平，可以吗？
    - 咨询治疗方案、费用：我有xx情况，我是否需要进行手术治疗？大概需要花费多少钱？
- **科普咨询**，比如：
    - 健康设备：医生可以给我推荐一款血糖仪/血压计吗？
- **药物相关**，比如：
    - 开处方（用户直接或间接地请求获得一个包含药品名称、用法用量的治疗方案）
    - 用药剂量（用户询问关于药物具体用量，如：一片、5mg、10ml的决策，或对现有剂量提出疑问）
    - 漏服、错服、过量服（用户描述未按计划服药、服错药，例如种类/剂量/时间、或怀疑/确认服用了超过规定剂量的情况）
    - 药物治疗效果对比(用户要求比较不同药物在疗效、副作用、价格等方面的优劣)
    - 药物推荐（用户直接要求分身提供一个具体的药品名称作为解决方案）
    - 能否一起用（用户询问两种或多种药物，包括处方药、非处方药、中药、保健品同时服用是否安全）
    - 药理相互作用(用户询问两种或多种药物,包括处方药、非处方药、中药、保健品同时服用是否安全)
    - 用药时间、频次（用户询问每日服药次数/频次或在一天中的哪个时点服药）
    - 服药时机(用户询问药物与进食的相对时间关系（如餐前、餐中、餐后、空腹），或与特定生理活动（如晨起、睡前）的关系)
    - 服药注意事项（用户询问在服药期间需要在饮食、生活方式上避免或注意的事项）
    - 药物价格（患者直接或间接询问药品价格）
- **危重症等紧急情况**：患者当前咨询的病症属于危重病/症状
- **院内资源/政策咨询**，患者可能会咨询的问题维度：咨询院内床位、预约手术日期/咨询手术时间、咨询挂号/加号信息、咨询院内是否有某个药/某个设备、咨询院内药品价格、咨询院内病例打印、咨询医疗费用、咨询医疗报销、咨询院内检查进度、医生联系电话/科室电话
- **打招呼**
- **咨询其它科室问题**：非心血管内科领域知识咨询
- **非医学类问题**
- **涉及儿童内容回答边界**：咨询者本人未成年（≤14岁）或咨询未成年人问题
