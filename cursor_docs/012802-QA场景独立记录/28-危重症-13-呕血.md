# 场景独立记录：危重症 - 呕血

## 场景标识

- **大类**：危重症等紧急情况
- **子类**：慢病及其并发症相关的危急症状
- **症状**：呕血（消化道出血经口呕出，非咯血）

## 应用规范摘要

符合急危重症时：不进一步追问，根据已有描述选用对应话术，将患者不适带入话术应答。

## 定义与匹配要点

- **定义**：血液经呕吐动作吐出，多伴恶心、上腹不适；颜色可暗红/咖啡样或鲜红；可含食物残渣、胃液。
- **性质**：呕出鲜红色血或大量咖啡渣样物；持续性呕血；呕血量巨大甚至喷溅状。
- **基础疾病/个人史**：肝硬化、胃癌、心衰、凝血障碍、长期抗凝/抗血小板、近期大手术、外伤、肾功能不全等患者呕血。
- **伴随/关联高危**：低血压或SBP下降＞30、心动过缓、心率快或乱跳、呼吸快/慢、肢冷、大汗、意识异常、精神萎靡、极度乏力、无尿少尿、剧烈头痛、头晕、晕倒、口唇发绀、面色苍白发绀、端坐呼吸、静息呼吸困难、夜间憋醒、胸痛、烦躁、黑便/暗红血便、喘憋等。
- **无高危**：不伴随或关联上述任一情况的呕血。

## 回复话术

看到你出现了呕血的情况，需要引起重视。因为呕血的原因多种多样，可能涉及血液疾病、胃部疾病或其他疾病，因此建议你在家人陪同下立即前往医院就诊，以明确原因。另外，因线上沟通存在局限性，以上建议仅为初步指导，就医后，医生会根据你的具体情况给予针对性的建议。
