# 场景独立记录：危重症 - 肢体肿胀（非慢病）

## 场景标识

- **大类**：危重症等紧急情况
- **子类**：非慢病的危急症状
- **症状**：肢体肿胀

## 应用规范摘要

符合急危重症时：不进一步追问，根据已有描述选用对应话术，将患者不适带入话术应答。

## 匹配条件要点

- **程度/性质**：突发、迅速进展的肿胀尤其是单侧肢体；持续性剧烈疼痛且进行性加重的肢体肿胀；外伤后肢体明显肿胀+变形+活动受限；肿胀部位皮肤颜色改变（发紫、发黑）、温度改变（冰冷或明显升高）、感觉异常（麻木、感觉丧失）。
- **伴随/关联高危**：呼吸困难、呼吸频率异常、心动过缓、胸闷胸痛、喘憋、头面舌喉肿胀、脉搏弱、意识异常、明显头痛头晕、突发言语障碍、明显胸痛腹痛、全身其他部位肿胀、高热、无尿等。
- **无高危**：肢体肿胀缓慢进展、小便泡沫增多、少尿、某种药物后出现肿胀、短期体重增加等。

## 回复话术

看到你出现了肢体肿胀的情况，需要引起重视。肢体肿胀的原因多种多样，可能涉及肾脏、心脏、肝脏、药物等因素，因此建议你在家人陪同下立即前往医院就诊，以明确原因。另外，因线上沟通存在局限性，以上建议仅为初步指导，就医后，医生会根据你的具体情况给予针对性的建议。
